# 提升效率的工具收藏

## 代码编辑器

### VS Code

Visual Studio Code 是我日常开发的主要编辑器，推荐以下插件：

- **Prettier** - 代码格式化
- **ESLint** - 代码检查
- **GitLens** - Git 增强

## 效率工具

### Notion

用于知识管理和笔记整理，支持：

- 数据库功能
- 模板系统
- 团队协作

### Obsidian

本地 Markdown 笔记工具，适合：

- 双向链接
- 知识图谱
- 离线使用

## 浏览器扩展

- **uBlock Origin** - 广告拦截
- **Dark Reader** - 深色模式
- **OneTab** - 标签页管理
